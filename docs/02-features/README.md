## Geospatial Features

### Google Maps Integration

- Address geocoding and validation
- Satellite imagery display
- Map-based location selection

### Roof Image Annotation

- Upload and annotate inspection photos
- Mark areas of concern
- Generate visual reports

### Satellite Image Workflow

- Retrieve satellite imagery for properties
- Overlay on reports
- Use for inspection planning
