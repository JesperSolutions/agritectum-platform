# Getting Started

Essential guides for developers new to the project.

## Quick Setup (15 minutes)

1. **Clone & Install**

   ```bash
   git clone <repo>
   cd agritectum-platform
   npm install
   ```

2. **Setup Firebase**
   - Copy `.env.example` to `.env.local`
   - Add your Firebase credentials
   - See [FIREBASE_SETUP.md](FIREBASE_SETUP.md)

3. **Start Development**

   ```bash
   npm run dev
   ```

4. **Login**
   - Email: `inspector@branch.local`
   - See test users in [LOCAL_DEVELOPMENT.md](LOCAL_DEVELOPMENT.md)
